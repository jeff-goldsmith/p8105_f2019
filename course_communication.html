Course Communication

Discussion board

The course’s discussion board is hosted on CourseWorks. The discussion board should be used to ask questions about course content – how code works, why it might not, and possible solutions. The teaching team will supervise the discussion board and respond regularly, but students are encouraged to be active in both asking and answering questions.

Some guidance on how best to ask questions on a discussion board can be found here and here.

Slack

We will use Slack for communication during lecture sessions. Each topic will have a slack channel (i.e. a discussion thread organized around a topic), which will be monitored during class. This is intended to provide real-time answers to questions in class, and will be used to convey solutions to in-class exercises. Slack channels are not intended to replace the discussion board on CourseWorks for more involved questions about content.

Sign up for the course Slack workspace here. You can use Slack from a web browser, or you can download an app to your Mac, Windows, Linux, Android, or iOS device. You can control whether you receive notifications on new posts by going to Preferences, as well as decide which ‘channels’ to subscribe to. You can learn more about Slack features here.
